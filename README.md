# Node-fsUtils

A library of file system helpers / extensions / utilities.

## Use 

Add to package.json :

    {
        {
            "dependencies": {
                "madscience-fsUtils"": "https://github.com/shukriadams/node-fsUtils.git#0.0.1"
        }
    }

Import

    let fsUtils = require('madscience-fsUtils');
